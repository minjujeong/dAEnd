UI Collection View

Collection View는 이미지 같은 데이터 아이템들의 레이아웃을 지정하는 역할을 한다. dAEnd 어플리케이션의 첫화면이 앨범인데, 어플리케이션 사용자들의 사진을 불러와 정렬하기 위해 본 UI가 필요했다. Data Sourcedhk Delegate 프로토콜을 갖는다는 점에서 Table View와 유사하며 둘 다 Scroll View의 하위 객체이지만 Collection View는 여기에 Layout Object까지 있다.
